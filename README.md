# 🚀 MyLink Project: 아이디어를 현실로 (바이브 코딩)
MyLink는 인스타그램 등 SNS 프로필에서 흔히 사용되는 "멀티 링크 페이지"를 나만의 스타일로 구현한 프로젝트입니다. 이 프로젝트는 구글 Antigravity IDE와 Gemini CLI를 활용한 '바이브 코딩' 방법론을 통해 개발되었습니다.

🛠 Tech Stack
Framework: Next.js

IDE: Google Antigravity

AI Assistant: Gemini CLI (Vibe Coding)

Language: TypeScript / JavaScript

Deployment: Vercel (또는 사용 중인 배포 환경)

✨ Key Features
Custom Link Tree: 인스타그램 프로필 링크처럼 여러 개의 링크를 하나의 페이지에 깔끔하게 정리.

Responsive Design: 모바일과 데스크톱 환경 모두에 최적화된 레이아웃.

Vibe Coding Implementation: Gemini CLI와의 대화를 통해 자연어 중심의 선언적 개발 진행.

💡 What is Vibe Coding?
이 프로젝트는 전통적인 방식의 코딩을 넘어, Gemini CLI를 활용해 개발자의 의도(Vibe)를 코드로 즉각 구현하는 바이브 코딩 방식을 채택했습니다.

Antigravity 환경에서 실시간 협업 및 코드 편집.

Gemini CLI를 호출하여 기능 구현, 버그 수정 및 리팩토링 수행.
